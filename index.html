How this works
1. Paste your Hosthub API key (Settings → API Keys → Create new API key)
2. Click Test connection — this discovers which endpoints respond on your account
3. Set the date range and click Fetch bookings
4. Click Load into app to replace the current booking data

Note: This requires running npm start (not just opening the HTML file) so the built-in proxy can forward requests to Hosthub without CORS issues.
